# Description
Show all job definitions.

# Usage

```bash
dcos job list [OPTION]
```

# Options

| Name, shorthand | Default | Description |
|---------|-------------|-------------|
| `--help, h`   |             |  Print usage. |
| `--info`   |             |  Print a short description of this subcommand. |
| `--json`   |             |  Print JSON-formatted list. |
| `--version, v`   |             | Print version information. |
