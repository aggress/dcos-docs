# Description
Shutdown a service.

# Usage

```bash
dcos service shutdown [OPTION]
```

# Options

| Name, shorthand | Default | Description |
|---------|-------------|-------------|
| `--help, h`   |             |  Print usage. |
| `--info`   |             |  Print a short description of this subcommand. |
| `--version, v`   |             | Print version information. | 

# Positional arguments

| Name, shorthand | Default | Description |
|---------|-------------|-------------|
| `<service-id>`   |             | The DC/OS service ID. |
